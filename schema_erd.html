Key Analytics Path

"Which sections fail most?"
requirement_results → JOIN requirements on requirement_id → GROUP BY section, code → COUNT by status

"Which projects have the most failures?"
requirement_results → JOIN reports on report_id → JOIN projects on project_id → GROUP BY project → COUNT failures

"Cost per org this month?"
run_pricing → JOIN reports on report_id → JOIN projects on project_id → JOIN organizations on organization_id → SUM estimated_cost
